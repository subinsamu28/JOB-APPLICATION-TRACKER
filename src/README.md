## üîß Data Structure

### **Application Object**

Each job application is stored as:

```typescript
{
  id: string;                    // Unique identifier
  company: string;               // Company name
  position: string;              // Job title
  status: string;                // Current status
  location: string;              // Job location
  salary?: string;               // Salary range (optional)
  jobType: string;              // Full-time, Part-time, etc.
  applicationDate: string;       // When you applied
  interviewDate?: string;        // Interview date (if scheduled)
  followUpDate?: string;         // When to follow up
  offerDeadline?: string;        // Offer decision deadline
  priority: string;              // Low, Medium, High
  contactPerson?: string;        // Recruiter name
  contactEmail?: string;         // Recruiter email
  jobUrl?: string;              // Job posting link
  notes?: string;               // Your notes
  cv?: Document;                // Uploaded CV
  coverLetter?: Document;       // Uploaded cover letter
  activities?: Activity[];      // Activity log
  interviewQuestions?: string[]; // Custom questions
}
```

### **How do I transfer data to another device?**
